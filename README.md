**Personal Portfolio Website** - A modern, interactive, and performance-optimized portfolio built with React and TypeScript.

## 🚀 Features

- **Interactive PCB Background**: A dynamic, animated Printed Circuit Board background.
- **Performance Optimized**: Implements **Lazy Loading** for all major sections to ensure fast initial load times.
- **Responsive Design**: Fully adaptive layout for desktop, tablet, and mobile devices.
- **Accessibility**: Enhanced with ARIA labels, keyboard navigation, and semantic HTML.
- **Modern Tech Stack**: Powered by **React 18**, **TypeScript**, and **Vite**.

## 🛠️ Tech Stack

- **Core**: React 18, TypeScript
- **Build Tool**: Vite

## 📦 Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/nolmedo/nolmedo.dev.git
   ```

2. Navigate to the project directory:

   ```bash
   cd nolmedo.dev
   ```

## 🏃 Usage

Start the development server:

```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) to view it in your browser.

## 🏗️ Build

To create a production build:

```bash
npm run build
```

## 🧹 Lint

Check for code quality issues:

```bash
npm run lint
```
